盲从

发表于 2013-05-27 | 分类于 Think | | 阅读次数

慢慢的，生活又变麻木了。

两个星期没有码字了，我总会在一件事上面越变越懒。今天星期一，星期三上午VB考试，真担心自己这样一直不复习会挂科，电脑上放着VB的复习资料，看也不想看。

慢慢地认识了“盲从”这个词的含义，大学不可以盲从，不可以同一个寝室别人做什么你就跟着做什么，因为甚至你们都不会思考为什么要这么做。和别人一起去上自习，和别人一起打DOTA，和别人一起作息……你做什么我便做什么，大家都一样。

就这样，开始变得毫无个性，一直的盲从变成了对生活的恐慌。

和别人开玩笑说：“屌丝都是注定孤独一生的。”其实，我们这些“屌丝”也许只有孤独，才能真正发现自己的潜力吧。

聚

发表于 2013-04-15 | 分类于 Live | | 阅读次数


本来应该昨天就写下这一篇的，但是由于种种原因好吧其实借口是昨天是我人生中18年喝的最多的一次，实在是睡意沉沉，hold不住扑向了枕头。

昨晚是和学姐第一次见面吃饭，和她很长时间没见到的高中同学一起，一共两位学长两位学姐。当然现在他们都是我的学长学姐了，一起聊天吃饭很开心。其间我被学姐的酒量吓到了学姐说这将是她一年中喝的最多的一次，并且多次向我申明她每年只会喝醉一次。学姐很健谈，对我说了很多话，不过一觉醒来能记起的只有零星片语了。

能记起的大概是几个叙述中学姐很佩服的人了。

他们的大学生涯可能没什么令人赞叹或者震惊的亮点，但是根据我自己的渣总结，他们所做的就是在大学里做自己以后再也不可能做的事情，并且始终如一地坚持下去。就比如一位学姐的室友，她可以在四年中每天在6：00-6：30之间起床，除了生病等可以原谅的因素，坚持叠被子（好吧这点我很惭愧，上学期也一直在叠，然后这学期，变懒了），每天锻炼一小时，虽然这并不能够体现什么，但是至少应该是我应该学习的。

听他们在四年后谈起自己的高中，有感动，有怀念。
