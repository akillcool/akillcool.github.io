不作死就不会死

发表于 2013-12-07 | 分类于 Think | | 阅读次数
想要写点什么，但是，这一段时间的我是真的迷茫了吗？四个字：随波逐流。

自制力越来越差了。每每这个时候，总是回想起高中时妈妈有一次骂我的时候说的两个字：贪婪。没错，就是贪婪。按照我的理解，就是习惯安逸并且想要不断地去追求更加安逸的生活。至今犹然记得高三时LION说的一句话：努力考个好学校吧，不然以后工作的安逸了，你是很难再去作出什么改变的。不会再去考研进修，不会再去考什么公务员（虽然现在我们对这个工作有很大的偏见但是如果有这个机会的话我想十个人中至少有五个会选择这个职业的吧），生活，以及以后的日子，就这么定下来了。

轻松的日子过得真快啊。这样的感觉体现在我对印象深刻的事情记忆的程度上，明明是两天前发生的事情，却感觉像是昨天发生的一样。做的有意义的事情太少了，生活的太安逸了，高三时的那种度日如年的感觉，充实的感觉，似乎都一去不复返了。

我讨厌这样没有自制力的自己。

就连写一篇心情，也是看到新勇在写什么才想起。他写的东西，一般都是要发到空间或者微信的吧。我不喜欢这样，不想让周围的人看到自己的心情，不喜欢被看穿的感觉。然后就自己发到没有熟人问津的博客上，作为纪念。

盲从

发表于 2013-05-27 | 分类于 Think | | 阅读次数

慢慢的，生活又变麻木了。

两个星期没有码字了，我总会在一件事上面越变越懒。今天星期一，星期三上午VB考试，真担心自己这样一直不复习会挂科，电脑上放着VB的复习资料，看也不想看。

慢慢地认识了“盲从”这个词的含义，大学不可以盲从，不可以同一个寝室别人做什么你就跟着做什么，因为甚至你们都不会思考为什么要这么做。和别人一起去上自习，和别人一起打DOTA，和别人一起作息……你做什么我便做什么，大家都一样。

就这样，开始变得毫无个性，一直的盲从变成了对生活的恐慌。

和别人开玩笑说：“屌丝都是注定孤独一生的。”其实，我们这些“屌丝”也许只有孤独，才能真正发现自己的潜力吧。

渐渐

发表于 2013-05-14 | 分类于 Live | | 阅读次数

Blog的主站貌似最近常常抽风，时不时无法访问，或者只能通过代理访问。

这周植物生物学实验要考试了，考的是各种各样的切片观察，据说是每个人随便抽一张，然后说出这张装片的结构就行了。每次五六个人一起去考试，然后据学长们说后面的人的装片全都和这五个人的一样，而我好死不死的学号是04，蛋疼了。还好班群里有好心的童鞋把装片的照片和结构说明做了一个DOC传了上来，看看还没自己看书来的详细，也就放心了。

然后说下Kindle，Kindle是4月8号到手的，到现在一个多月了，感觉自己的看书习惯有了很大的改变，Kindle在床上不好躺着看，双手举着很累，困了就直接锁屏睡觉，而且Kindle最让我欣慰的就是一个星期充一次电就够了。关于看的书，爱看豆正在改版，改版之前从里面推送了不少英文书，可惜的是到目前为止一本都没有坚持看完，总是会看着看着就会感觉很疲劳，然后去看轻松的小说去了。后来把Kindle里面的什么小说都删光了，觉得太无聊了又拖进去了一本《锦衣夜行》。希望自己可以慢慢坚持看英文书籍的习惯，静下心来看书。

上个星期葛少年生日，中午请了大家一起吃麻辣香锅，晚上去KTV腐败，确实花了不少钱。到了晚上12点回宿舍楼的时候发现平常会给晚点回的同学们留一道门缝的宿管阿姨把门给锁死了，无处可去的我们只好跑去网吧陪小胖包夜，打了一晚上的守卫剑阁，连跪一晚上。第二天体育课要测一千米，只有我和江少年去了，他倒数第五，我倒数第三，然后回来打DOTA到下午3点，一直睡觉到晚上九点。

葛少年的生日礼物是给自己买的网球拍，上周去练了一次，今天下午索性直接在网球场打了，还别说网球场还真抢手，再晚来5分钟就没场子打了，然后就是两个菜鸟相互练习发球的节奏了～～～不过还是打的很开心，能运动真好。
