若是心里长了太多草

2014-07-27 17:48
life
约 817 字 预计阅读 2 分钟

这两天因为手头有了大哥给了买手机的赞助费，不打算买手机，计划买一台Win8平板。比较来比较去终于还是看中了戴尔的Venue 11 Pro打算入一台。自己海淘的话时间周期太长，半个暑假就过去了于是打算去某宝买已经从官网开始shipping的订单。

嗯，Venue 11 Pro大概有如下四种配置（LTE版本除外）

① Z3770               +   2G内存   +   64G SSD

② Core i3 4020Y  +   4G内存   +   128G SSD

③ Core i5 4210Y  +   4G内存   +   128G SSD

④ Core i5 4300Y  +   8G内存   +   256G SSD

奸商们的报价都很黑，当然我打算买官翻退订机，新货实在是壕的选择，另外有那钱不看性价比的话干脆直接上苏菲婆3算了何必像现在这样纠结得要死。

可能是因为想了太多心里长了太多草，脑子一抽直接下单了一个顶配的机器，现在想来真是作死的选择。

8G内存用来做甚?在10.8寸的触摸平板上开虚拟机么？

256G SSD用来做甚?对一个手持设备来说空间已经足够了，就算是iPad顶配也才128G，又不是一个32G的平板装完系统就只剩15G左右的空间了，况且以我多年做好人修电脑的功底，删掉英文版的Windows恢复分区重新上一个纯净专业版系统不是难事，128G这样下来应该还有接近100G可以用。

至于CPU，抱歉我不觉得4300Y比起4210Y来说多出的那些超频有什么用，反正核芯显卡都一样。最后还是用了顶配的CPU (/ω＼)

于是乎前天下了单，昨天在激动兴奋中度过一天，晚上却辗转反侧一晚上睡不着好吧其实是因为我的一天是从中午开始的所以一晚上不睡大概没有什么问题。想来想去还是觉得去买顶配是一件无比逗比的事——那贵出来的1K块可以在某宝买多少零食啊，还可以海淘把平板的所有贵的像奇葩一样的配件给配齐了。

于是，我刚刚做出了一个非常艰难的决定，决定趁卖家还没发货趁我还有理智赶紧退货买128G版的。

就是这样。

款还没退，我还不能睡。

Update: 退货成功

Update2：下了一个i5 4210Y 4G 128G版本的单

Update3：店家说他订错了单，于是讨价还价一番加了一百块，CPU变成了顶配版的i5 4300Y
