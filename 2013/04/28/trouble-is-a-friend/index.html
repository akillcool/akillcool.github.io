Trouble is a Friend

发表于 2013-04-28 | 分类于 Live | | 阅读次数

这个星期，做出了一点改变。

希望能够坚持下去。

首先是坚持每天起床去食堂吃饭，其次…好吧这个“其次”让父上母上看到一定会打断我的腿不可 就是我终于认真的听了一节有机化学课。

前两天看到父上企鹅微博上发了这样一段话：

你若希望掌控未来，那就必须把握现在！如果有人要击倒你，那只能说明你比他们更优秀！多数的错与失，是因为不努力，不坚持，不挽留，然后告诉自己，一切都是命运！

好吧，看到这个其实我不知道该说什么好，或者说该怎么说，“把握现在”真的不容易，很轻易的就把时间过去了——比如没事的时候刷刷煎蛋看看暴漫，刷刷草榴看看推特，时间是以小时为单位度过的，比如上课带着kindle成上午或成下午的看，比如晚上上床关灯后还会再看个半小时到两小时。

中午午睡醒来，流了很多口水，睡的很舒服。然后听到的是床下室友打游戏一连串不断的鼠标点击声。好吧我承认我开始厌倦这样的生活了，只能在心中悄悄的写下：

I hate what I‘ve become.

然后去洗个冷水澡。

Trouble will find you no mater where you go

No Matter if you’re fast no matter if you’re slow

The eye of the storm and the cry in the morn

Your fine for a while but then start to loose control…
